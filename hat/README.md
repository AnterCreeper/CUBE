# CUBE HAT

### Overview

Extension HAT for CUBE, enable Audio and HID function.

### Feature

- Everest-semi ES8156 Audio DAC with Headphone Driver
- WCH CH583M BLE MCU
- MS5351M Clock Generator with 22.5792MHz Crystal(support 44.1kHz and 48kHz)
